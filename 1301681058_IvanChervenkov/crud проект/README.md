<h1> Моят Crud проект с login опция </h1>

Моят проект е база данни, която е свързана към конзола на MS Visual Studio 2013.
За да стартирате и тествате проекта трябва последователно да изпълните стъпките, които са описани по-долу.

<h2> Стартиране на проекта </h2>

1.Стартирайте Microsoft Visual Studio и по-точно Crud solution от папката "crud проект".
2. След като сте изпълнили стъпка едно, отидете на менюто Build и от него натиснете Build Solution.
3. Трябва да изпълните sql файла users, а това става чрез по-долните подточки --->
a) Първо трябва да изпълните само първия ред на кода (create database[users])
б) Вторият ред от базата данни, след което може да execute-те останата част от заявката
4. Отидете във VS и променете в класа UserRepository следния стринг "Server=VANKO-PC\\VANKO;Database=users;Trusted_Connection=True;";"
със следното " @"Data Source=FMI-532-0\SQLEXPRESS;Initial Catalog=users;User ID=sa;Password=sa;Integrated Security=false" "
5. ВНИМАНИЕ !!! Ако има съществуваща база с име users, да бъде изтрита и точка 3 да бъде изпълнена.
6. Вече имате пълната свобода да стартирате проекта и да разгледате какво съм направил.
